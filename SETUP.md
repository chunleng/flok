# Setup

The steps to setup your environment to develop Flok is documented here. For
installation, please refer to [README.md](./README.md)

## Prerequisite

- (Required) Rust: version documented in ./.tool-versions)
- (Optional) ASDF: version manager for Rust and other tools

## Common Commands

```bash
# Run with ./.flok.json
cargo run

# Run with custom configuration
cargo run -- -c ./tests/assets/flok.json
```
